# 🤖 Interview AI

An AI-powered web application that analyzes your **resume**, **job description**, and **self-description** to generate a detailed **interview preparation report** and a professional **resume** — tailored just for you.


## ✨ Features

- 📄 **Resume Analysis** — Analyzes your existing resume against the job description
- 💼 **Job Description Matching** — Identifies skill gaps and strengths
- 🧠 **Self-Description Processing** — Takes your personal summary into account
- 📊 **Interview Report Generation** — Generates a detailed interview preparation report
- 📝 **Resume Generation** — Creates a professional, job-tailored resume


## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React.js, Vite |
| Backend | Node.js, Express.js |
| Database | MongoDB Atlas |
| AI | Gemini API |


## 🚀 Getting Started

### Prerequisites

- Node.js v18+
- MongoDB (local or Atlas)
- npm or yarn


## 🔮 How It Works

1. User uploads or pastes their **Resume**
2. User enters the **Job Description**
3. User writes a short **Self Description**
4. AI analyzes all three inputs
5. App generates:
   - ✅ Personalized **Interview Report** with likely questions & tips
   - ✅ Optimized **Resume** tailored to the job
